# Flappy Haskell

Flappy Bird implementation made with SDL2 and FRP (Yampa).

![](bird.gif)

## Installation

You will need some system libraries:
 - SDL2
 - SDL2_Image
 - SDL2_Mixer (with libvorbis/Ogg support enabled!)

Don't forget to get the submodules then:

```
stack build
stack exec flappy-haskell
```
